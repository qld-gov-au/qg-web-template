Implementation

Queensland Government Web Template (web template) is build to comply with the IS26 Guidelines.

To help with rapid development, the web template now comes integrated with Bootstrap 3.

Usage

There are two main ways to use the QGW Template:

  1. For Franchises; directly implemented
  2. For Agencies; Customised (in compliance with IS26)

Franchises: direct implentation of the template

After downloading the template page and assets developers should be able to mostly follow bootstrap standard styling guidelines to produce correctly styled pages.

Agencies: Customised template

The Queensland Government Website Template has bootstrap 3 baked into it's core. In most instances, you can create content using bootstrap styles, and it will output apropriate Queensland Government Website styling.
