Asides

Custom

An aside represents a section of a page that contains secondary content that is designed to be complementary to the main content (e.g. related initiatives, latest news, upcoming events, contact details, tips, customer stories or case studies) and remains meaningful when separated from the main content. Publishers may add an aside to any page with a right-hand panel.

Please refer to the SWE Content Page Template for examples of asides.
