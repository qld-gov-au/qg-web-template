Dropdowns

Customised

Due to the nature of the Forms system being used, dropdowns are not advised for web forms.

The Template standard for implementing dropdowns is using the Queensland Government icon. To enable this, add the class qg-icon to your button. 


<button class="btn btn-default dropdown-toggle qg-icon"
  type="button"
  id="dropdownMenu"
  data-toggle="dropdown"
  aria-haspopup="true"
  aria-expanded="true">
    Dropdown
</button>

For more information on implementation, see the bootstrap dropdowns section.
